## Описание

Приложение «Movie Library» предназначено для ведения личной коллекции фильмов. Программа позволяет добавлять фильмы с указанием названия, жанра, года выпуска и рейтинга, просматривать список в таблице, фильтровать записи по жанру и году, а также сохранять данные в JSON-файл и загружать их при следующем запуске.

**Основные возможности:**

- Добавление нового фильма с проверкой корректности ввода
- Отображение списка фильмов в таблице
- Фильтрация по жанру (частичное совпадение)
- Фильтрация по году (точное совпадение)
- Автоматическое сохранение данных в файл movies.json
- Загрузка данных из JSON-файла при запуске
